# 基本

元素：由一对标签括起来的东西

属性：元素所具有的属性，可以有布尔属性（input type="text" disabled）



### 元素分类

#### 块级元素

p, h*, header等, 被认为是一个块(block)

#### 内联元素

em, strong, a等, 需要闭合标签

#### 空元素

img, br等, 不需要闭合标签



### 头部元素

#### 标题 title

title是整个html文档的标题

* 注意，h1是用来标记页面内容的标题

#### 数据 meta

charset=“utf-8”

name&content

* author: Lvris
* description: A personal blog for ......
* keywords: code...(这玩意已经被搜索引擎忽略了)

#### 链接 link

* rel用于指定作用 “shortcut icon” “stylesheet”(似乎古老)
* href指定位置 “http://...”

#### 样式 style

* 内部调用样式表

#### 脚本 script 

* 用src指定位置(source)



### 文本元素

#### 特殊样式

| 标签   | 作用             | PS   |
| ------ | ---------------- | ---- |
| strong | 重要, 一般加粗   | b    |
| em     | 强调, 一般倾斜   | u    |
| del    | 删除, 一般删除线 |      |
| ins    | 更新, 一般下划线 |      |
|        |                  |      |
|        |                  |      |
|        |                  |      |

#### 标题 h*

* 只使用一次h1
* 每页层级不超过三个

#### 列表 *l

* 使用ul与ol来表示无序和有序列表，使用li来表示项目
* 使用dl来表示描述列表(项目dt, 描述dd)，用于专业术语的陈述

#### 引用 quote

* blockquote与q
* 使用cite说明来源

#### 缩略 abbr

* 使用title进行解释

#### 总结 summary

- 放在details中间
- h5的新元素

#### 样式元素

colgroup: 用类似列表的方式来让col元素指定几列(span)的样式

#### 层级元素

thead, tbody, tfoot等, 没有添加会默认为tbody

注意: 层级元素要在colgroup之后

# 链接

### URL

用于定义在网络上的位置, 格式如下

protocol://hostname[:port]/path/[;parameters]\[?query]#fragment

#### 协议类型

| 协议          | 作用           | PS             |
| ------------- | -------------- | -------------- |
| http          | 超文本传输协议 |                |
| https         | 安全版本(ssl)  |                |
| ftp           | 文件共享       |                |
| file          | 本地文件       |                |
| mailto        | 邮件发送       | 会调用邮箱软件 |
| ed2k/thunder… | P2P下载        | 需要bt下载软件 |

#### 确定位置

- 使用“/” “../”访问上下级目录
- 隐去域名 /: 服务器顶部根域名 
- 隐去协议 //: 使用相同协议

#### 额外参数

* 使用“?…&…”来提供一些额外参数
* 使用“#”来访问具有特定id属性的部分



### 实例

#### 访问别的网页

```html
<a href="http://lvris.com" title="qwq">
    <strong>Blog</strong>
</a>
<a href="http://lvris.com" title="qwq">
    <img src="*.png" alt="这里应该有一张图">
</a>
```

#### 提供下载链接

```html
<a href="https://download.mozilla.org/?product=firefox-latest-ssl&os=win64&lang=zh-CN"
   download="firefox-latest-64bit-installer.exe">
  下载最新的 Firefox 中文版 - Windows（64位）
</a>
```

#### 发送电子邮件

```html
<a href="mailto:nowhere@mozilla.org?cc=nobody&bcc=nobody&subject=sub&body=body">
  Send mail with cc, bcc, subject and body
</a>
```
