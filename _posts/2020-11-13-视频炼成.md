### 视频

#### 帧 Frame

##### FPS

视觉惰性(persistence of vision)

闪烁频率：45.8Hz才会觉得不闪

胶片24格：平衡了音频损耗和成本

国内的帧率一般是**25**

游戏的帧速率无法保持稳定,从而显得不够流畅

##### I帧

帧内压缩时的参考帧,与原画中割大抵是不同的概念

#### 分辨率 Resolution

HD：720P 1080P

FHD：1920*1080

UHD：2K, 4K

#### 长宽比 Aspect ratio

| 比例    | 小数     | 用途     |
| ------- | -------- | -------- |
| 4:3     | 1.333... | 学院标准 |
| sqrt(2) | 1.414... | 国际纸张 |
| 3:2     | 1.5      | iphone4  |
| 16:10   | 1.6      | 宽屏电脑 |
| 1:phi   | 1.618... | 黄金比例 |
| 5:3     | 1.666... |          |
| 16:9    | 1.777... | 电脑常用 |

#### 制式 NTSC/PAL

|      | 电视线 | 场数 | 场序    | FPS    | 分辨率  |
| ---- | ------ | ---- | ------- | ------ | ------- |
| NTSC | 525    | 60   | 偶 - 奇 | 29.97i | 720*480 |
| PAL  | 625    | 50   | 奇 - 偶 | 25i    | 720*576 |

时代变了，模拟电视转向数字电视

#### 码率 Bit Rate

##### 类型

CBR(Constant Bit Rate)

VBR(Variable Bit Rate)

##### 压制

crf是x264编码器的一种参数,越大码率越低,一般网络播放24即可

2pass就是扫两遍,便于更合理控制码率

#### 色彩

##### 色深/位深

表示色彩的种类，一般的8/10bit表示位深

##### 色度采样

(YUV)Y为亮度，其他两种表示色差(CbCr)

采样格式用444/422/420这样的数字表示

分别代表:第一行色度采样数,第二行色度采样数

#### 格式

##### 编码

- 有帧内(intra)压缩和帧间(inter)压缩之分

| 编码                | 备注                           |
| ------------------- | ------------------------------ |
| H.264(AVC/MPEG4P10) | x264为免费好用的算法，H264商用 |
| H.265(HEVC)         | 未普及                         |
| Prores(APPLE)       | 苹果下使用帧内压缩             |
| Cineform(Gopro)     | 微软下使用帧内压缩             |

##### 封装

| 格式 | 备注                                                        |
| ---- | ----------------------------------------------------------- |
| AVI  | 古老而巨大的格式，Xvid编码比较小                            |
| RMVB | 低码率下的选择，不利于剪辑，RV40相当小                      |
| WMV  | 微软开发，可以DRM加密(兼容性很差)                           |
| MP4  | Moving Picture Experts Group-4 最为常见,支持h264            |
| MKV  | 开源而广泛的格式，能封装软字幕和大量音轨                    |
| MOV  | Apple开发的格式，带透明通道，转格式的时候可以考虑(兼容性好) |
| FLV  | 来自flash，网络传播广泛，F4V=FLV+H.264                      |

### 音频

#### 听觉

##### 频率 - 音调

20Hz~20KHz可以被识别

##### 振幅 - 响度

##### 波形 - 音色

##### 基音 + 泛音

基音决定音调，泛音决定音色

#### 码率

采样频率：常用44.1KHz（历史） 48KHz

采样精度：bit计,常用192kbps

#### 环绕

##### 声道

单声道 mono

双声道 Stereo立体声

##### 音箱

2.0 两个卫星音箱

2.1 超重低音subwoofer，无指向性

4.1 前后置音箱

5.1 LR前置 LSRS环绕 C中置人声

#### 格式

有损：mp3 wma

无损：wav ape flac

(现在基本都是AAC编码)

### 杂项

#### 动画片源

TV：台标 时间 滚动 圣光暗牧

BD：蓝光 一般画质更好

* Director’s cut 可以理解为完整版
