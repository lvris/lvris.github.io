## 活用

### 基本活用推导

| 修饰/连体 | い             | な               |
| --------- | -------------- | ---------------- |
| 陈述/终止 | い             | だ               |
| 副词/连用 | く             | に               |
| て形/连用 | くて           | で               |
| ~礼貌     | いです         | です             |
| 否定      | くない         | じゃない         |
| ~礼貌     | ~くないです    | じゃないです     |
| 过去/连用 | かった         | だった           |
| ~礼貌     | かったです     | でして           |
| 过去否定  | くなかった     | じゃなかった     |
| ~礼貌     | くなかったです | じゃなかったです |

* 在转换的时候把いい看成よい



#### な形容词「～い」

綺麗「きれい」

嫌い「きら」

幸い「さいわ」
