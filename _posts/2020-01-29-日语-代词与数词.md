## 代词

### 称谓

#### 第一人称

| 用词           | 谦敬            | 性别       |
| -------------- | --------------- | ---------- |
| 私「わたし」   | 敬              |            |
| 私「わたくし」 | 敬，正式        |            |
| 僕「ぼく」     | 敬(古) - 平(今) | 男性对平辈 |
| 私「あたし」   | 平              | **女**     |
| 俺「おれ」     | 平              | **男**     |

#### 第二人称

| 用词(后缀) | 谦敬 | 性别 |
| ---------- | ---- | ---- |
| さん       | 敬   |      |
| 君「くん」 | 平   | 男   |
| ちゃん     | 平   | 女   |
| あんた     | 怒   |      |

一般用对方的名字加上合适的头衔。

像是「お前、手前、貴様」这种一般只会出现在漫画里)
